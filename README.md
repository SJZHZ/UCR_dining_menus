# UCR Dining Menus Crawler

## Information
A Python automation script that crawls [UCR FoodPro](https://foodpro.ucr.edu/foodpro/search.aspx) to search for specific keywords in dining menus and sends notifications via Slack, Telegram, or Lark.

All generated by AI with intelligent code assistance and automation tools.

This project was created using [Kwaipilot](https://kwaipilot.ai/) and [Cline](https://cline.bot/) to demonstrate AI-assisted development workflows.

## Installation

**Recommended: Use conda for environment management**

1. **Create Conda Environment**
   ```bash
   conda create -n menu python=3.9
   conda activate menu
   ```

2. **Install Dependencies**
   ```bash
   pip install -r requirements.txt
   ```

2. **Create Configuration**
   - Copy `config_template.json` to `config.json`
   - Edit `config.json` with your notification credentials

3. **Set up Keywords**
   - Edit `keywords.txt` to add/remove search keywords
   - Each line is a separate keyword

## Usage

### Quick Start
```bash
# Run the complete workflow
python run.py
```

### Crontab Setup
Add this line to your crontab to run the crawler daily at 8 AM:
```bash
0 8 * * * cd /path/to/UCR_dining_menus && python run.py
```

For conda environment:
```bash
PATH=/home/ubuntu/miniconda3/bin:/usr/local/bin:/usr/bin:/bin

0 8 * * * cd /path/to/UCR_dining_menus && conda run -n menu python run.py
```


## Configuration

### Setting up Notifications

- Slack
  1. Create a Slack webhook in your workspace
  2. Set `"enabled": true` and provide the webhook URL

- Telegram
  1. Create a bot using [@BotFather](https://t.me/BotFather)
  2. Get your bot token and chat ID
  3. Set `"enabled": true` and provide credentials

- Lark (Feishu)
  1. Create a bot in your Lark workspace
  2. Get the webhook URL
  3. Set `"enabled": true` and provide the webhook URL

### Keywords File Format

Each line in `keywords.txt` represents a search keyword:

```txt
brisket
ribs
```

Lines starting with `#` are treated as comments.

## Dependencies
- `requests`: HTTP requests
- `beautifulsoup4`: HTML parsing
- `lxml`: XML/HTML parser
- `python-dotenv`: Environment variables
- `slack-sdk`: Slack notifications
- `python-telegram-bot`: Telegram notifications
- `aiohttp`: Async HTTP client
- `asyncio-throttle`: Rate limiting
